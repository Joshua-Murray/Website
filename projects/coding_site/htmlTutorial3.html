Images

11
The img tag embeds an image into your HTML. Always found with the 'src' attribute, which tells the browser where to find the image. Note that the img tag is self-closing.
Syntax
<img src='mylocalimage.jpg'/>

Line breaks

12
This tag is used in a block of text to force a line break. This is to be used for content which is a single paragraph, but where this formatting is necessary such as poems or addresses. To separate paragraphs, separate each paragraph into a separate <p> element instead. The resulting element on a web page will look like:
Example
<p> Some text <br/>
that spans two lines </p>

Links

13
Link elements are used to connect your document to a related resource. Link elements appear only in the head section of a document so they do not alter the content, but only the presentation. Link elements are most commonly used to connect to a stylesheet, script, favicon, or alternate format of the page such as an RSS feed or PDF
Example
<link type="text/css" rel="stylesheet" href="styles.css" />

Lists

14
HTML supports two kinds of lists: ordered lists and unordered lists. Within lists each individual list item has its own tag.

Unordered Lists

Unordered lists are just lists whose items are denoted with bullet points.
Example
Shopping list

<ul>
<li>Dish soap</li>
<li>Kitty litter</li>
<li>Tomato sauce</li>
</ul>

Ordered Lists

15
Ordered lists items are denoted with numbers.
Example
My numbered list

<ol>
<li>First item!</li>
<li>Second item!</li>
<li>Last item!</li>
</ol>
